# base Gulp & Fractal

## Install

` npm install `

## Usage

### develop mode

` gulp `

- Dart Sass compiler
- Browser sync start
- Babel
- watch

### build

` gulp build `

- sass compile
- styleguide build

make ` ./styleguide ` directory

### style guide view

` gulp styleguide `

- styleguide Browser sync start
